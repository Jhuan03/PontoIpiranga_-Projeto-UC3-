# PontoIpiranga_-Projeto-UC3-
Site desenvolvido como projeto da UC3 utilizando React. O objetivo é ajudar meu pai, que tem um pequeno ponto em frente de casa onde realiza vários serviços para levantar uma renda extra. Este site funciona como uma página informativa sobre o Ponto Ipiranga, apresentando os serviços oferecidos e facilitando o acesso da comunidade.
